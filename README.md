# Software Packaging Project

## Project Overview

This project demonstrates software packaging using Node.js and Express. It covers dependency management, environment configuration, Semantic Versioning, packaging, and security auditing.

---

## Dependencies

- Node.js
- Express
- dotenv
- nodemon

Install dependencies using:
npm install

---

## Installation Commands

Clone the repository:
git clone https://github.com/omodunnialice30-blip/software-packaging.git

Navigate into the project:
cd software-packaging

Install dependencies:
npm install

---

## Versioning Strategy

This project follows Semantic Versioning (SemVer).

Version used:
1.0.0

- Major version – breaking changes
- Minor version – new features
- Patch version – bug fixes

---

## Packaging Commands

Initialize project:
npm init -y

Install Express:
npm install express

Install dotenv:
npm install dotenv

Install nodemon:
npm install --save-dev nodemon

Start the application:
npm start

---

## Security Audit

Security audit was performed using:
npm audit

Any vulnerabilities reported were reviewed and resolved where applicable.

---

## How to Run the Application

Start the application:
npm start

Open your browser and visit:
http://localhost:3000
